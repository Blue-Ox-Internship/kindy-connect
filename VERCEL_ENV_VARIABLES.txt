# ============================================
# TROUBLESHOOTING
# ============================================

If deployment fails:
- Check that all variables are added correctly
- Make sure no extra spaces in values
- Verify all three environments are selected
- Check Vercel deployment logs for errors

If database connection fails:
- Check Supabase dashboard - database might be paused
- Click "Restore" if paused
- Wait 2-3 minutes and redeploy

If login fails:
- Run the SQL to create admin user in Supabase
- File: supabase/migrations/20250109_add_quick_superadmin.sql
